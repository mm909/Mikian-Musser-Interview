I am more than a Resume.

Teaching is my purpose.

Fall 2018 — Code Central, Las Vegas

At Code central I work with kids ages 7-17 and teach them how to code! I teach the younger kids how to program using blocks (Scratch/Code.org) and older kids how to program in javascript, HTML, CSS, P5.js, and Gamemaker language. I also host classes focusing on advanced programming topics like robotics, genetic algorithms, AI, or physics.

Summer 2018 — iD Tech, Rice University

I taught visual programming, Lego Mindstorms, and Scratch to classes of eight kids ages seven to ten years old. I created lesson plans, homework assignments, and coding challenges for kids with variable math and coding skill levels. I balanced teaching the class and working with each student one on one to ensure each student was learning and felt included.
